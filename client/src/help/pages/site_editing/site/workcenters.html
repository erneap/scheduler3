Site Workcenter Editing

(Site Scheduler/Leadership only)

The workcenter is the intermediate level of separation at the site. It is assumed that the site will have an admin or leads workcenter, plus two or three other workcenters. The overall site schedule is produced by workcenter. Each workcenter can be created with multiple shifts and/or positions. The shift is a designated work period for multiple employees (example: Day shift might cover the period from 6am to 4pm.) In contrast, a position is usually assigned to an individual, regardless of the work period.

  1. Add a New Workcenter
    1. Select "Add New Workcenter" from the list under "Workcenters". The display will change and show two fields for editing, the workcenter's identifier and name or title, both are required.
    2. Type in the identifier for the workcenter. This is normally a short string to use as an identifier. It will be saved in lower-case letters.
    3. Type in the display name or title for the workcenter.
    4. Click the "Add" button to save the workcenter at the end of the current list. The "Clear" button can be clicked before adding to remove all data from the two fields.
  2. Edit Workcenter
    1. Select the workcenter you want to edit. The only field editable is the workcenter's display name/title.
    2. Type in the changes to the name/title you need to make. The changes will be saved as you exit the field.
  3. Changing the Workcenter Display Order
    1. Select the workcenter whose order you want to move up or down. The arrows to move up/down the list will display depending on the position of the workcenter within the list.
    2. Click the Up or Down Arrow button to move the workcenter up or down the list. After clicking the button, the new display order will be shown.
  4. Delete a Workcenter
    1. Select the workcenter you want to delete.
    2. Click the red trashcan icon above the list to start the process. A deletion dialog will be displayed. Click the "Yes" bottom to complete the deletion, or "No" button to stop the process.
  5. Long Lists for Workcenters, Positions or Shifts - If the list is larger than the size of list view. Use your mouse scroll function to move the list up or down. The outer up/down pane is for sorting the list, not scrolling it.
  6. Add a Position to a Workcenter
    1. Select the workcenter from the workcenters list you want to add the new position to.
    2. The workcenter's shifts list will normally be displayed at this point. Click the "Workcenter Position" tab to display the position list for the workcenter.
    3. Select "Add New Position" from the list. Two fields will be displayed just like for a new workcenter. These are position identifier and name/title.
    4. Type in the new position's identifier and name/title. After the values are entered, a "+" button will display to the right of the name/title field.
    5. Click the "+" button to add the new position. The list will display showing the new position at the bottom of the list.
  7. Edit a Position within a Workcenter
    1. Select the workcenter from the workcenters list you want to edit one of its positions.
    2. The workcenter's shifts list will normally be displayed at this point. Click the "Workcenter Position" tab to display the position list for the workcenter.
    3. Select the workcenter position you want to edit. Two fields will be displayed with current data, the name/title field plus a list of employees assigned to this position.
    4. If necessary, edit the name/title for the position. The value will be saved upon exit from the field.
    5. Multiple employees can be assigned to a position. Click on the field to display the list of possible employees. Use the checkbox next to each employee name to be assigned. The list will be saved after checking or unchecking the each checkbox.
  8. Add a Shift to a Workcenter
    1. Select the workcenter from the workcenters list you want to add a new shift.
    2. The workcenter's shifts list will normally be displayed at this point. If not, Click the "Workcenter Shifts" tab to display the shift list for the workcenter.
    3. Select "Add New Shift" from the list. Two fields will be displayed just like for a new workcenter. These are shift's identifier and name/title.
    4. Type in the new shift's identifier and name/title. After the values are entered, a "+" button will display to the right of the name/title field.
    5. Click the "+" button to add the new position. The list will display showing the new position at the bottom of the list.
  9. Edit a Shift within a Workcenter
    1. Select the workcenter from the workcenters list you want to edit one of its shifts.
    2. The workcenter's shifts list will normally be displayed at this point. If not, Click the "Workcenter Shifts" tab to display the shift list for the workcenter.
    3. Select the shift you want to edit. Four fields will be displayed to allow you to edit the shift's information. These include its name/title, associated work codes, any pay code and the minimum number of employees that should be working on the shift to consider it minimumally covered. Enter each field as necessary: (all fields save after exiting it.)
      1. Type in the modified name/title.
      2. You can select multiple work codes that are to be associated with this shift. The work code list is provided by the team.
      3. The shift may have a pay code associated with it. This is normally associated with different pay differentals for working on this shift. This is defined by the accounting system.
      4. Site leadership is responsible for insuring a minimum number of employee's are working on the shift. The "Minimum Working" field is this minimum number.
    4. Type in the new shift's identifier and name/title. After the values are entered, a "+" button will display to the right of the name/title field.
    5. Click the "+" button to add the new position. The list will display showing the new position at the bottom of the list.
  10. Change Position/Shift Display Order
    1. Select the workcenter from the workcenters list you want to edit one of its shifts/positions.
    2. The workcenter's shifts list will normally be displayed at this point. If you want to change the order for displaying a position, Click the "Workcenter Positions" tab to display the position list for the workcenter, otherwise, click the "Workcenter Shifts" tab to display the shift list.
    3. Select the shift or position you want to change its display order.
    4. Click either the up or down arrow button to move the selected shift or position in the direction pushed. The change will display after the change is saved to the database.
  11. Delete Position/Shift
    1. Select the workcenter from the workcenters list you want to edit one of its shifts/positions.
    2. The workcenter's shifts list will normally be displayed at this point. If you want to change the order for displaying a position, Click the "Workcenter Positions" tab to display the position list for the workcenter, otherwise, click the "Workcenter Shifts" tab to display the shift list.
    3. Select the shift or position you want to delete.
    4. Click the red trashcan icon to the far right to start the deletion process. A dialog will display to verify the deletion, click the "Yes" button to continue and delete it or "No" to cancel the process.
