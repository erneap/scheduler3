Employee's Leave Editor

(Site Scheduler/Leadership only)

This display/editor will display all the employee's leave (taken or approved) for a single calendar year.

  1. Adjust the Displayed Calendar Year
    1. Click the "<<" button to adjust the viewing calendar backward one year.
    2. Click the ">>" button to adjust the viewing calendar forward one year.
  2. Edit a Leave Day
    1. Find the date you want to edit. All four fields can be editted unless the leave status is already actual. Actual leaves are either previously so marked or were verified by ingest data. All updated fields will be saved upon exit from the field.
    2. If Leave Date is to be edited - The leave date must be in the form MM/DD/YYYY to be saved.
    3. If you want to change the Leave Type - A list of available leave codes, select the new leave code from the list provided.
    4. If you want to change the Leave's Hours - The hours are in decimal hour format and must contain a minimum of one digit to the right of the decimal place.
    5. If you are changingin the Leave's Status - The leave status can be one of three values (Requested, Approved or Actual). Select one of these from the list provided.
  3. Add a Leave Day - At the bottom of the display four input fields are present to all the new leave. All four fields must be entered before you will be able to add it to the employee's leave list.
    1. The leave date must be in the format "MM/DD/YYYY". Type in the new leave date.
    2. Select the leave code to be used from the list provided.
    3. Type in the number of hours for the leave in decimal hour format with a minimum of one digit to the right of the decimal point.
    4. Select one of the three values (Requested, Approved, or Actual) for the leave status.
    5. After all four values are entered or selected, a "+" sign will be visible on the left of the entry fields. Click the "+" sign to add the new leave. It will be added to the list of leaves.
  4. Delete a Leave Day
    1. Find the leave you need to delete.
    2. Click the delete button to the left of the leave you want to delete. A verification dialog will be displayed to verify the deletion. Click the "Yes" or "No" button to verify the deletion or cancel the deletion.
