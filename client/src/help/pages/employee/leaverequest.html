Leave Request Editor

Requesting Leave of any type starts with this portion of the Scheduler application.

(NOTE: As an example to follow, the steps are provided to request a vacation over about a two-week period, but will consist of several holidays used in conjunction with the normal vacation days.)

The leave request editor opens as the blank slate show above, with the possible exception of having previously request leaves shown in the "Current Leave Request" list.

  1. Drafting a New Leave Request:
    1. Click on the "Request Dates" field's calendar icon on the right side. A calendar will display to allow you to select the inclusive dates for the leave. These must include regular days off, so that if your work schedule assignment changes the leave can be recalculated. Click on the first and last days of this period within the calendar. It should highlight the period selected.
    2. Next, select the "Primary Leave Code" to be used for the majority of the leave days displayed. You can select the leave code to use on individual days later.
    3. Lastly, use your mouse and click the "+" button to the right of the "Primary Leave Code" selector. This will send this basic information to the server and display the request for further editing, as shown below.
    4. Now this leave request is in "draft" mode and is only in the DRAFT state. If you want to change some of the days to Holidays, you can edit individual days by going to that day and changing the type of leave to use/project for that day and the number of hours. You can make changes to any leave request until the leave is actually taken, even if it has been approved.   2. Clearing the Selected Leave Request - Once you choose a leave request to display, the application thinks you are editing that leave request. So to clear off the display and get back to the "Add" mode, click on the Clear button ( ) to the right of the "Primary Leave Code" to clear the selected leave request.
